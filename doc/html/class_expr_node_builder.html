std::list< Exception > & exps )
static

Функция создания дерева из строки разбора выражения

Функция принимает строку в обратной польльской нотации и строит по ней дерево логического выражения. В случае выявления ошибок в записи дерева, ошибку добавляются в список exps. Если после exps.size() != 0, то функция возвращает nullptr

Parameters
[in]postfixExpr- строка разбора логического выражения
[in,out]exps- список выявленных ошибок при построение дерева
Returns
указатель на корень дерева или nullptr, если при построении дерева возникли ошибки
