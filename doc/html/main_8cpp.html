int main ( int argc,

Точка входа в программу.

Функция проверяет аргументы командной строки и запускает программу в соответствующем режиме. Если аргументы соответствуют режиму тестирования, запускаются тесты. В противном случае запускается основная программа с переданными путями к входному и выходному файлам.

Note
При argc=2 и argv[1]=--test - запускаются тесты
При argc=3 - запускается основная программа, где параметры:
argv[1] - путь к входному файлу с разбором логического выражения, записанного в постфиксном формате
argv[2] - путь к выходному файлу, где будет содержаться html код таблицы истинности
Parameters
argcКоличество аргументов командной строки.
argvМассив аргументов командной строки.
Returns
int Код возврата. 0 - успешное завершение, 1 - ошибка.

Запуск программы из консоли для Windows.
Для запуска тестов:

ModuleWork_Q_RS_main.exe --test # запускает тесты

Для запуска основной программы:

ModuleWork_Q_RS_main.exe input.txt out.html
