double ExprNode::calculate ( )

Метод вычисляет арифметическое значение выражения.

Precondition
Перед вызовом необходимо проверить вычисляемое ли значение выражения арифметически с помощью метода ExprNode::isConstantExpr. При попытке вызвать данный метод у дерева, значение которого невозможно вычислить арифметически, будет выброшено исключение ArithmeticCalculationException.

void ExprNode::getParameters ( std::vector< ExprNode * > & params)
virtual

Метод выполняет поиск параметров логического выражения

Метод начинает поиск параметров с левой ветки в узле, после переходит на правую. При нахождение параметра узлу присваивается ID и он (если ещё не находится в params ) добавляется в выходной списко params

Parameters
[in,out]params- заполняемые найденными параметрами список
