Функция преобразует строку в десятичное число.

Эта функция принимает строку, представляющую число в любом из поддерживаемых форматов в С++ (двоичный, восьмеричный, десятичный, шестнадцатеричный) и преобразует её в десятичное число с плавающей точкой. Формат числа определяется автоматически на основе содержимого строки.

template<typename T >
T reverseBits ( T num,
size_t nb = sizeof(T) * CHAR_BIT )

Функция разворачивает заданное количество битов в числа.

Эта функция принимает число и разворачивает указанное количество бит, начиная с младшего бита.

Template Parameters
Tтип числа, для которого производится операция.
Parameters
numчисло, биты которого нужно развернуть.
nbколичество бит, которые нужно развернуты (по умолчанию, все биты в числе).
Returns
число с развернутыми битами.
Exceptions
std::invalid_argumentесли количество бит для переворота превышает количество бит в числе.
Note
Если nb не указано, по умолчанию используется количество бит в числе типа T.

Пример использования:

uint8_t x = 0b00001111;
uint8_t y = reverseBits(x, 8); // y будет равно 0b11110000
