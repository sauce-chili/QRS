Реализация бинарной коммутативных операций

Реализация бинарной коммутативных операций Отличие от BinaryArithmeticNode заключается в том, что при сравнение с другими коммутационными опециями проверяется коммутация корня выражения. Коммутационными операциями являются: EXPR_NODE_TYPE::MUL (*), EXPR_NODE_TYPE::PLUS (+), EXPR_NODE_TYPE::XOR (^), EXPR_NODE_TYPE::EQ (==),EXPR_NODE_TYPE::NEQ (!=)

Пример коммутационных операций: a + b = b + a; a * b = b * a; a ^ b = b ^ a; a == b = b == a; a != b = b != a; Более сложные пример: (5 + x) * a = a * (5 + x), однако (5 + x) * a != (x + 5) * a
